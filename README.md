# OpenBeePdfSplitter

## Description

Application Windows permettant de découper automatiquement un PDF
à partir du texte OCR :

Num Commande Open Bee

## Fonctionnalités

- OCR automatique
- Découpage PDF
- Renommage automatique
- Journal CSV
- Interface graphique PySide6

## Dépendances

Python 3.12+
